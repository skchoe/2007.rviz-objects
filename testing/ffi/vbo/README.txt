7 differents tests are available. To launch a specific test 
start an execurable with a parameter between 1 and 7 
included.

1: Vertex Buffer Object as Vertex Array
2: Indexed VBO
3: Interleaved VBO
4: Serialized VBO
5: Vertex Mapping
6: VBO with GLSL
7: VBO with Cg

Projects are available for VC6.0, VC7.1 and VC8.0. A mingw 
version is included but it requires to be fixed.
